9. Define and Describe is Structured Query Language(SQL)?
It’s a standardized language for interacting with RDBMS.
10. List and Describe each of the different subsets of SQL(Mean DDL, DML, DCL, TCL)?
Ddl: data definition language that creates, alter, truncate, drop and rename
database. It’s also auto-committed.
Dml: data manipulation language that inserts, updates, selects, and delete on
the stored data, which isn't auto committed.
Dcl: data control language that can give access or revoke to the database.
Tcl: transaction control language uses to manage changes that's made by dml.
