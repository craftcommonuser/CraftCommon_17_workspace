1.   what is Data?

3.      what is Database(DB)?
	Storage of organized data that can be accessed, updated, deleted and managed.

 
4.      What is the Relation Database Management System(RDBMS)?
	A system used to maintain relational databases.


5.      Define the importance of Relation Database Management System(RDBMS)?

7. 	List examples of Relation Database Management System(RDBMS)?

8.       List examples of Non-Relational DB(Nosql)? 
		MangoDB, OrientDB


9.       Define and Describe is Structured Query Language(SQL)?
		It’s a standardized language for interacting with RDBMS.


10.       List and Describe each of the different subsets of SQL(Mean DDL, DML, DCL, TCL)?
		Ddl: data definition language that creates, alter, truncate, drop and rename database. It’s also auto-committed.
		Dml:  data manipulation language that inserts, updates, selects, and delete on the stored data, which isn't auto committed.
		Dcl: data control language that can give access or revoke to the database.
		Tcl: transaction control language uses to manage changes that's made by dml.

11.      what is table in Database(DB)?

12.     what is column and Row(tuples) in table?
		
		row contains information that describes a single entity while data in a column describes a field of information all entities possess.
