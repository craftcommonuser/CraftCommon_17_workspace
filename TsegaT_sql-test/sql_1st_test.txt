9.       Define and Describe is Structured Query Language(SQL)?
     The core of a relational DB which is used for accessing and managing the DB.
     SQL is standard language to interact with RDBM and mainly perform create, read, update, and delete

10.       List and Describe each of the different subsets of SQL(Mean DDL, DML, DCL, TCL)?
DDL(data definition language):it make change in to structure of the table by create, alter, truncate, drop and rename) and it is auto-committed 
 DML(data manipulation language) : used for manipulating the data stored in the table. By insert, update, delete, and select.
DCL(data control language) grants or revokes data accesses permitions.
TCL(transaction control language) used to manage the data transaction by DML statement. we can commit, rollback, savepoint, and set transactions.
